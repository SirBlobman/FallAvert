# FallAvert

A plugin that prevents some actions while a player is in the air.

## Requirements

- Java 8
- [Spigot](https://www.spigotmc.org/), [Paper](https://papermc.io/downloads/paper), or [Folia](https://papermc.io/software/folia) 1.8.8-1.19.4
